Для расчетов резьбы на прочность используем следующую методику:
	/****************/
1. This books -
1.1 Расчеты и конструирование деталей машин. Левятов (зеленая для ПТУ) стр.69
1.2 Детали машин. (серенькая) Добровольский стр.134
1.3 Расчеты на прочность деталей машин. Биргер стр.37
1.4 Резьбовые соединения. Учебное пособие. Тарханов стр.6 
	/****************/
2. Материалы для резьбовых деталей. (по Детали машин. (серенькая) Добровольский стр.134)
2.1 Принимаем допустимое напряжение на разрыв
2.1.1 для углеродистой стали [sigma]p <= 0,4*[sigma]текучести (или 0,25*[] для болта менее М16)
2.1.2 для легированой стали  [sigma]p <= 0,3*[sigma]текучести (или 0,20*[] для болта менее М16)
	/****************/
Делаем расчеты в двух вариантах - для стандартного болта и гайки с классом прочности по ГОСТу
и для нестандартной детали из произвольно указанной стали.
	/****************/
Класс прочности болтов обозначен двумя числами.
Первое число, умноженное на 100, определяет минимальное значение [сигма]временное, МПа;
Второе число, умноженное на 10, отношение [сигма]текучести/[сигма]временное,%
т.е. произведение чисел, умноженное на 10 - определяет величину предела текучести в МПа.
	/****************/
